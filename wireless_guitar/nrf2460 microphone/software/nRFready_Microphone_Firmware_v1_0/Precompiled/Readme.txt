These HEX files has been compiled by using :
- MPLAB IDE V8.73
- HI-TECH C Compiler for PIC10/12/16 MCUs (Lite Mode) V9.82

HEX files available:
- Receiver.hex
- Microphone_Blue.hex
- Microphone_Red.hex

Open AdjustableParameters.h and uncomment the following line to compile for RED microphone:
//#define RED_MIC   //uncomment this to create .hex file for red mic. Otherwise it will be the blue mic.
